LSTM

The Long Short Term Memory (LSTM) funtion was trained on several tickers to estimate the highest and lowest price within the next 15 trading days based on the recent trading prices. It will also produce a buy/sell recomendation.

Required: Base URL

Base URL: "http://site/query?"

Required: Function Name

Function Name: "function=LSTM"

Required: Ticker

Stock Ticker Symbol (i.e. "symbol=A")

Optional: Date to Predict

Date in MM-DD-YYYY format (i.e. "date=01-01-2021")
