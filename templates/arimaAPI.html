ARIMA

The ARIMA model attempts to predict the next day's closing price based on the stock's price history.

Required: Base URL

Base URL: "http://site/query?"

Required: Function Name

Function Name: "function=ARIMA"

Required: Ticker

Stock Ticker Symbol (i.e. "symbol=A")

Optional: Date to Predict

Date in MM-DD-YYYY format (i.e. "date=01-01-2021")

Full request:

http://site/query?function=ARIMA&symbol=A&date=01-01-2021
