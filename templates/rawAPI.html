Raw Data

The Raw Data function returns the daily open, high, low, close, and volume of the selected ticker symbol. If the extended option is selected, then the additional indicators and buy/sell triggers are returned as well.

Required: Base URL

Base URL: "http://site/query?"

Required: Function Name

Function Name: "function=raw"

Required: Ticker

Stock Ticker Symbol (i.e. "symbol=A")

Optional: Extended Dataset

Whether or not to return the additional indicators and triggers (i.e. "extended=True")

Full request:

http://site/query?function=raw&symbol=A&extended=true
