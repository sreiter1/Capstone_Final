Update Database

The "Update Database" funtion will update the data of a particular stock. The source of the data has a limit of 5 calls per minute and 500 calls per day. If the stock symbol has data through the most recent trading day, then this funciton will not update the ticker. This function will aslo calculate several triggers and indicators associated with the ticker symbol.

Required: Base URL

Base URL: "http://site/query?"

Required: Function Name

Function Name: "function=update"

Required: Ticker

Stock Ticker Symbol (i.e. "symbol=A")

Full request:

http://site/query?function=update&symbol=A
